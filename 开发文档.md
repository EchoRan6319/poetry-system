# 🏆 “传承经典”诗词大赛双屏互动系统 - 开发设计与实施文档

> **版本**：V3.0  
> **技术支持**：信息工程与大数据学院  
> **架构特征**：纯前端架构 / 一机双屏 / 零后端部署 / 数字新国风 UI  

---

## 一、 项目概述

本项目专为全校性古诗词比赛设计，由**信息工程与大数据学院**提供核心技术支持。系统采用首创的**“一机双屏·免后端”**纯前端架构，结合“数据山水”的数字新国风 UI 设计，旨在将传统的古诗词背诵转化为兼具电竞感与国家级晚会质感的现场互动游戏，同时在全校师生面前低调且优雅地展现理工科学院的技术实力。

---

## 二、 核心技术栈选型

本项目完全摒弃了传统的“前端+后端+数据库+WebSocket”的重度架构，采用纯前端本地化方案，极大降低了部署成本和现场网络不稳定带来的风险。

| 模块类别 | 选用技术 | 核心优势与解决的痛点 |
| :--- | :--- | :--- |
| **基础框架** | Vue 3 + Vite | 极速构建，响应式系统完美契合比赛状态机（计分、倒计时、题目流转）。 |
| **样式引擎** | Tailwind CSS | 实用优先，极速还原大字号、高对比度的大屏排版与毛玻璃（Glassmorphism）特效。 |
| **跨屏通讯** | `BroadcastChannel` | 浏览器原生 API，实现**控制台与大屏的纯前端本地秒级同步，完全免除 WebSocket 和后端服务器开发**。 |
| **跨端渲染** | `Noto Color Emoji` | 强制使用谷歌开源彩色字体，完美解决 Windows 投屏下 Emoji 降级为黑白简笔画的问题。 |
| **数据管理** | 本地 JSON 题库 | 通过大模型 AI 智能体生成标准题库结构，前端 `fetch` 读取，即改即用。 |

## 三、 UI/UX 设计规范：数字新国风 (Data Landscape)

在视觉表达上，系统巧妙融合了“大数据科技感”与“中国传统文化”，通过以下设计语言实现跨界碰撞：

* **色彩基调：** 深渊墨色（`zinc-900`）为主背景，辅以青绿（象征科技与生机）与赤金（象征传统与尊贵）。
* **材质纹理：** 极简的数据网格（隐喻数据矩阵）叠加半透明拟态卡片，卡片边缘配以古典折角线条。
* **字体对撞：** 诗词内容使用端庄的**宋体/明体**（传递文化底蕴）；倒计时、比分等数字使用极细的**无衬线科技字体**（彰显极客感）。
* **品牌植入：** 屏幕右下角设置极具极客质感的专属低调水印：`Powered by College of Information Engineering and Big Data`，做到克制且优雅的炫技。

## 四、 六大核心题型设计

所有题目均基于初高中必背篇目，利用 AI 逻辑引擎精准生成，满足不同维度的考核需求：

1. **Emoji 猜诗：** 剥离意境，严格遵循“逐字对应”或“关键象形/谐音”逻辑（如：☀️🌙🚶🚶 -> 两个黄鹂鸣翠柳）。
2. **宫格寻诗：** 在 9宫格/12宫格 中打乱原诗，并混入极具迷惑性的同音/形近干扰字，考验瞬时拆解能力。
3. **易错找茬：** 选取高中教材公认易错字（如“樯橹”、“沧海一粟”），制造单字错误供抢答。
4. **中原风华：** 河南地域文化特色题，聚焦河南籍诗人或描写中原名胜的诗作。
5. **经典填词：** 聚焦生僻字或难写字（如：嫠、咽、砯），配合大屏倒计时进行限时书写。
6. **主题飞花令：** 围绕“颜色、动物、数字、地名”等意象进行车轮战抢答。

## 五、 数据驱动规范 (JSON Schema)

前端题库统一采用以下标准化 JSON 结构进行解析与渲染，保证数据与视图的完美解耦：

```json
[
  {
    "id": "唯一标识符 (String)",
    "stage": "题型分类 (String, 如: Emoji猜诗)",
    "rule": "大屏展示的规则与限时说明 (String)",
    "question": "题目内容实体 (String, 支持Emoji串/宫格串/带错字诗句)",
    "answer": "标准答案 (String)",
    "key_point": "导播台主持人备忘录/易错点提醒 (String)",
    "time_limit": "倒计时秒数 (Number, 如: 8)"
  }
]
```

## 六、 敏捷开发实施路径 (Sprint Plan)

- [ ] 搭建 Vue 3 + Vite 工程结构，配置 Tailwind CSS。

- [ ] 针对 6 大题型开发独立的 Vue 渲染组件，处理 Emoji 散开、九宫格排列等特殊排版逻辑。
